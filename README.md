# Khatabook Clone

> 📖 **Business hua easy** - A static UI clone of the Khatabook desktop landing page.

## ✨ Overview

This project is a high-fidelity frontend clone of the popular [Khatabook](https://khatabook.com) landing page. It showcases core web development skills in designing modern interfaces using semantic HTML5 and vanilla CSS. The layout includes structured sections like the hero area, app highlights, detailed feature breakdowns, and a comprehensive footer, beautifully recreating the original user experience.

## 🚀 Features

- **Semantic HTML5 Structure**: Well-organized code using modern HTML5 tags.
- **Custom CSS Styling (`khatabook.css`)**: Fully stylized without the reliance on external CSS frameworks.
- **Responsive Navigation**: Includes the brand logo, contact information, and a Call-To-Action (Log In) button.
- **Hero Section**: Eye-catching design with a mobile/desktop device mockup, interactive input fields, and gradient shapes.
- **Feature Modules**: Highlights the application modules including *Bahi Khata* management, payment reminders, and the BizAnalyst integration.
- **Desktop & Mobile Mockup Containers**: CSS-based mockup placeholders to showcase how the app runs across platforms.
- **Detailed Footer**: Complete and structured footer carrying necessary company/legal links, mirroring the original site.

## 🛠️ Technology Stack

- **HTML5**: For structure and content rendering.
- **Vanilla CSS3**: Used for robust styling, flexbox/grid layouts, gradients, and typography.
- **Zero Dependencies**: Lightweight, built completely from scratch with zero framework overhead.

## 💻 Getting Started

To view the project locally, you don't need any complex setup or a build process.

1. **Clone the repository** (if hosted on GitHub):
   ```bash
   git clone <your-repo-url>
   ```
2. **Navigate to the directory**:
   ```bash
   cd Khatabook-clone
   ```
3. **Open `index.html`**: Simply double-click on `index.html` to open it in your default web browser. For a better development experience, you can use the **Live Server** extension in VS Code.
